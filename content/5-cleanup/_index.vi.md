#### Xóa stack bằng AWS CloudFormation console

1. Mở bảng điều khiển [AWS CloudFormation](https://console.aws.amazon.com/cloudformation)

2. Trong trang **Stack** của **AWS CloudFormation**, chọn **Stack** mà bạn muốn xóa
3. Chọn **Delete**
4. Chọn **Delete stack**


#### Xóa Repository

1. Mở trang [Amazon ECR console](https://console.aws.amazon.com/ecr/repositories)

2. Trên thanh điều hướng, chọn **Region** chứa repository cần xóa
3. Chọn **Repository**
4. Trong trang **Repository**, chọn **Private**
5. Chọn **Repository** cần xóa và chọn **Delete**
6. Xác minh xóa **Repository** và chọn **Delete**


#### Xóa EKSCluster

1. Mở trang [Amazon EKS console](https://console.aws.amazon.com/eks/home#/clusters)

2. Chọn cluster cần xóa và chọn **Delete**
3. Xác minh xóa cluster và chọn **Delete**

#### Xóa môi trường AWS Cloud9 bằng AWS Cloud9 console

1. Chọn **AWS Cloud9** và chọn **Go TO Your Dashboard**
2. Chọn môi trường cần xóa và chọn **Delete**
3. Xác thực xóa môi trường bằng cách điền ``Delete`` và chọn **Delete**. Môi trường sẽ được xóa sau vài phút.
