#### Thực hiện fork Source Code

1. Mở link [GitHub](https://github.com/duocntg1/aws-guide-resume-template) 

- Chọn **Fork**

![Fork Source Code](/images/8-Forksourcecode/0001-forksourcecode.png)

2. Trong giao diện fork

- Kiểm tra **Repository name**
- Nhập **Description**, nhập ```fork aws-guide-resume-template```
- Chọn **Create fork**


![Fork Source Code](/images/8-Forksourcecode/0002-forksourcecode.png)


3. Giao diện sau khi tạo fork


![Fork Source Code](/images/8-Forksourcecode/0003-forksourcecode.png)

4. Trong giao diện **Github**

- Chọn vào ảnh đại diện gốc phải 
- Chọn **Settings**

![Fork Source Code](/images/8-Forksourcecode/0004-forksourcecode.png)

5. Trên thanh điều hướng, chọn **<> Developer settings**

![Fork Source Code](/images/8-Forksourcecode/0005-forksourcecode.png)

6. Trong giao diện **Developer settings**

- Chọn **Personal access tokens**
- Chọn **Generate new token**

![Fork Source Code](/images/8-Forksourcecode/0006-forksourcecode.png)

7. Trong giao diện tạo **personal access token**

- Nhập **Note**: ```aws-guide-eks```
- Chọn **repo**

![Fork Source Code](/images/8-Forksourcecode/0007-forksourcecode.png)

8. Chọn **Generate token**

![Fork Source Code](/images/8-Forksourcecode/0008-forksourcecode.png)

9. Sao chép **Personal access token**

![Fork Source Code](/images/8-Forksourcecode/0009-forksourcecode.png)
