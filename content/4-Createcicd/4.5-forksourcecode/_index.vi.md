#### Thực hiện fork Source Code

1. Truy cập vào trang [Github cá nhân](https://github.com/)

![Fork Source Code](/images/8-Forksourcecode/0001-forksourcecode.png)

2. Truy cập vào repository cần fork tại [First-Cloud-Journey/000062-EKSCICD](https://github.com/First-Cloud-Journey/000062-EKSCICD)

![Fork Source Code](/images/8-Forksourcecode/0002-forksourcecode.png)

3. Trong giao diện **Repository**

- Chọn **Fork**

![Fork Source Code](/images/8-Forksourcecode/0003-forksourcecode.png)

4. Thực hiện các bước để **Create a new fork**

- **Repository name**, nhập ```000062-EKSCICD```
- Chọn **Create fork**

![Fork Source Code](/images/8-Forksourcecode/0004-forksourcecode.png)


5. Giao diện sau khi fork source code

![Fork Source Code](/images/8-Forksourcecode/0005-forksourcecode.png)

6. Trong giao diện **Repository**

- Chọn vào **avatar** của **Github**
- Chọn **Settings**

![Fork Source Code](/images/8-Forksourcecode/0006-forksourcecode.png)


7. Trong giao diện **Settings**

- Chọn **<>Developer settings**


![Fork Source Code](/images/8-Forksourcecode/0007-forksourcecode.png)

8. Trong giao diện **Developer settings**

- Chọn **Personal access tokens**
- Chọn **Generate new token**


![Fork Source Code](/images/8-Forksourcecode/0008-forksourcecode.png)

9. Trong giao diện **New personal access token**

- **Note**, nhập ```awsfcj/000062```
- **Expiration**, chọn 30 ngày
- Phần **Select scopes** chọn **repo**

![Fork Source Code](/images/8-Forksourcecode/0009-forksourcecode.png)

10. Chọn **Generate token**

![Fork Source Code](/images/8-Forksourcecode/00010-forksourcecode.png)

11. Sau khi tạo token xong thì sao chép lại để sử dụng các bước tiếp theo

![Fork Source Code](/images/8-Forksourcecode/00011-forksourcecode.png)

12. Hoàn thành quá trình fork source và tạo **personal access tokens**

![Fork Source Code](/images/8-Forksourcecode/00012-forksourcecode.png)
