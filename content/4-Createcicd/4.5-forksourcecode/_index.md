#### Implement Source Code fork

1. Go to [Personal Github](https://github.com/)

![Fork Source Code](/images/8-Forksourcecode/0001-forksourcecode.png)

2. Access the fork repository at [First-Cloud-Journey/000062-websource](https://github.com/First-Cloud-Journey/000062-websource)

![Fork Source Code](/images/8-Forksourcecode/0002-forksourcecode.png)

3. In the **Repository** interface

- Select **Fork**

![Fork Source Code](/images/8-Forksourcecode/0003-forksourcecode.png)

4. Take the steps to **Create a new fork**

- **Repository name**, enter ```000062-websource```
- Select **Create fork**

![Fork Source Code](/images/8-Forksourcecode/0004-forksourcecode.png)


5. Interface after fork source code

![Fork Source Code](/images/8-Forksourcecode/0005-forksourcecode.png)

6. In the **Repository** interface

- Select **avatar** of **Github**
- Select **Settings**

![Fork Source Code](/images/8-Forksourcecode/0006-forksourcecode.png)


7. In the **Settings** interface

- Select **<>Developer settings**


![Fork Source Code](/images/8-Forksourcecode/0007-forksourcecode.png)

8. In the **Developer settings** interface

- Select **Personal access tokens**
- Select **Generate new token**


![Fork Source Code](/images/8-Forksourcecode/0008-forksourcecode.png)

9. In the **New personal access token** interface

- **Note**, enter ```awsfcj/000062```
- **Expiration**, choose 30 days
- Section **Select scopes** select **repo**

![Fork Source Code](/images/8-Forksourcecode/0009-forksourcecode.png)

10. Select **Generate token**

![Fork Source Code](/images/8-Forksourcecode/00010-forksourcecode.png)

11. After creating the token, copy it to use the next steps

![Fork Source Code](/images/8-Forksourcecode/00011-forksourcecode.png)

12. Complete the source fork process and generate **personal access tokens**

![Fork Source Code](/images/8-Forksourcecode/00012-forksourcecode.png)
